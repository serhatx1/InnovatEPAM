# Quickstart

## Prerequisites

- Node.js 20+
- Supabase project (URL, anon key, service role key)

## Setup

1. Install dependencies: `npm install`
2. Create `.env.local` with Supabase credentials.
3. Run dev server: `npm run dev`
4. Apply SQL schema/RLS in Supabase.

## MVP Validation

1. Register and log in.
2. Submit an idea with one file.
3. View idea in listing/detail.
4. Log in as admin and accept/reject with comment.
